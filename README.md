## **Setup & Installation**  
1. **Clone the repository**  
   ```bash
   git clone https://github.com/Nikhil-Info/port-v1.git
   cd port-v1
   ```

2. **Run locally**  
   - Open `index.html` in any modern browser (Chrome/Firefox/Edge)  
   - Or use Live Server extension in VS Code
